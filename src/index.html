about
Spaghetti Tonnetz is an open source tonnetz diagram viewer built for browser in javascript using p5.js and p5.sound libraries.

This project is licensed under the GNU General Public License v3, the source code can be found here.

The software is built using:
  • p5.js - JavaScript library for creative coding [LGPL v2.1]
  • p5.sound - Audio addon for p5.js [MIT License]
  • webmidi.js - Web MIDI API helper [MIT License]
  • Montserrat Project Font - Very cool font [SIL OFL v1.1]
